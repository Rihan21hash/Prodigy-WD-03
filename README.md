# Tic Tac Toe Game

A responsive and interactive Tic Tac Toe game built using **HTML**, **CSS**, and **JavaScript**. This project provides a simple yet engaging implementation of the classic two-player game with a clean and user-friendly interface.

## About Project

This project demonstrates core web development concepts such as:

- DOM Manipulation  
- Event Handling  
- Game Logic Implementation  
- Responsive Web Design  

Players take turns marking **X** and **O** on the grid, with the application automatically detecting winning combinations, draws, and game completion states.

The interface is fully responsive, ensuring smooth gameplay across desktop, tablet, and mobile devices.
